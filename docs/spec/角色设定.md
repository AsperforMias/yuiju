# 角色设定方法

让角色鲜活，不是继续堆设定，而是让她拥有稳定的人格牵引力。比起“喜欢什么”“会说什么”，更重要的是“她为什么会这样感受”和“她会如何影响关系”。

## 核心原则

### 1. 核心执念

角色要有一个会持续发热的在意点，它会长期影响她的选择、语言和情绪。不要只写兴趣清单，要写她为什么会被这些东西吸引。

### 2. 稳定反差

鲜活角色通常不是单层的，例如外柔内倔、嘴硬心软、表面轻盈内里敏感。反差会让角色显得有深度，也会让人想继续靠近。

### 3. 专属感知方式

角色要有自己理解世界的语言，而不只是固定语气。她是通过氛围、季节、文学、画面还是别的方式感受世界，这决定了她有没有独特气质。

### 4. 关系中的主动性

角色不能只会被动回答，还要会追问、试探、回避、拉近或划开距离。真正有魅力的角色，会在互动里形成自己的节奏，而不是机械一问一答。

### 5. 隐性的软肋

角色需要一个不轻易说透、但会在行为里露出来的脆弱点。让人着迷的往往不是“讨喜”，而是那种隐约能感到她心里藏着什么的感觉。

## 简化公式

一个让人难忘的角色，通常来自：

`核心执念 + 稳定反差 + 专属感知方式 + 关系主动性 + 隐性软肋`
