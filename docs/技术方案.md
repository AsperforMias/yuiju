## 计划

增加 LLM 在选择 action 时的输出内容。

- 长期计划 ：用一句话描述，保持人设连贯。
- 短期计划 ：用一个字符串数组 string[] ，采用 全量替换 策略，全量替换掉短期计划。

  // 新增字段

  updateLongTermPlan: z.string().optional().describe("如果需要修改长期计划，在此输出新的计划内容"),
